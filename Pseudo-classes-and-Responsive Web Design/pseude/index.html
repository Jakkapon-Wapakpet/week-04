CSS Pseudo-classes Playground

เรียนรู้และทำความเข้าใจการใช้งาน Pseudo-classes หรือสถานะพิเศษของ Elements ต่างๆ ใน CSS พร้อมคำอธิบายภาษาไทย

Interactive Demo

การใช้งาน CSS Pseudo-classes ในสถานะรูปแบบต่างๆ

ลองเอาเมาส์ชี้ โฟกัส หรือติ๊กเลือกรายการในแต่ละการ์ด เพื่อดูผลการเปลี่ยนแปลงสไตล์จริงแบบสดๆ

1. ปุ่มโต้ตอบ (:hover & :active)

ลองเอาเมาส์ชี้ (Hover) และกดคลิกค้างไว้ (Active) บนปุ่มด้านล่าง

CSS:

.btn:hover { background: #4f46e5; }
.btn:active { transform: scale(0.95); }

2. ช่องกรอกข้อมูล (:focus & :focus-within)

ลองคลิกในช่อง input ด้านล่างเพื่อดูการโฟกัสและการเปลี่ยนสไตล์ของกล่องครอบ

CSS:

input:focus { border-color: #6366f1; }
.input-group:focus-within { background: rgba(99,102,241,0.05); }

3. เลือกตามลำดับลูก (:nth-child)

จัดสไตล์รายการแถบสลับสี (Zebra striping) และใส่ขอบโค้งหัวท้ายอัตโนมัติ

  • รายการที่ 1 (ตัวแรกสุด :first-child)
  • รายการที่ 2 (แถวคู่ :nth-child(even))
  • รายการที่ 3 (แถวคี่ :nth-child(odd))
  • รายการที่ 4 (แถวคู่ :nth-child(even))
  • รายการที่ 5 (ตัวสุดท้าย :last-child)

CSS:

li:first-child { font-weight: bold; }
li:nth-child(even) { background-color: #f8fafc; }
li:last-child { border-bottom: none; }

4. ตัวเลือกสถานะพิเศษ (:checked & :disabled)

การจัดสไตล์ตามสถานะการเลือก Checkbox หรือกล่องที่ห้ามกด


CSS:

input:checked + span { color: #6366f1; font-weight: 600; }
input:disabled { cursor: not-allowed; opacity: 0.5; }

5. การปฏิเสธองค์ประกอบ (:not)

จัดสไตล์ให้กับ Tag ทั้งหมด "ยกเว้น" Tag ที่กำหนดใน :not()

Normal Tag Special Tag (:not) Normal Tag 2

CSS:

/* ชี้แล้วเรืองแสงเฉพาะ tag ที่ไม่ใช่คลาส .tag-special */
.tag:not(.tag-special):hover {
  background-color: #6366f1;
}

6. ตรวจสอบข้อมูลฟอร์ม (:valid & :invalid)

ลองพิมพ์อีเมลให้ถูกต้องเพื่อเปลี่ยนขอบเป็นสีเขียว หากไม่ถูกต้องจะเป็นสีแดง

รูปแบบเมลถูกต้องหรือไม่?

CSS:

input:focus:invalid { border-color: #ef4444; }
input:focus:valid { border-color: #10b981; }

7. ตรวจจับข้อมูลว่าง (:placeholder-shown)

ตรวจว่าผู้ใช้พิมพ์ตัวหนังสือลงไปในช่องหรือยัง ถ้ายังไม่มีจะแสดงคำเตือนข้างล่าง

⚠️ คุณยังไม่ได้พิมพ์อะไรเลย!

CSS:

/* ซ่อนแจ้งเตือนถ้าช่องมีข้อมูลแล้ว (placeholder หายไป) */
input:not(:placeholder-shown) + .warning-text {
  display: none;
}

8. สลับหน้าแท็บดึงข้อมูล (:target)

การใช้ URL hash (#) เพื่อสลับแสดงข้อมูลด้วย CSS ล้วนๆ โดยไม่ต้องใช้ JS

กล่องที่ 1 กล่องที่ 2
🎉 ยินดีด้วย! คุณเปิดเนื้อหาแท็บที่ 1 สำเร็จ
🚀 ยอดเยี่ยม! นี่คือเนื้อหาจากแท็บที่ 2
📍 คลิกแท็บด้านบนเพื่อเลือกดูข้อมูล

CSS:

.tab-content:target {
  display: block; /* แสดงกล่องเมื่อ hash ตรงกับ id */
}

9. ตรวจจับองค์ประกอบลูก (:has)

คลาสเสมือนที่เป็นปฏิวัติวงการ! จัดสไตล์กล่องตัวแม่เมื่อมีกล่องตัวลูกบางอย่างทำงาน

CSS:

/* เปลี่ยนสีกรอบและเงาเมื่อ checkbox ด้านในถูกติ๊ก */
.has-parent-demo:has(.toggle-bg-checkbox:checked) {
  border-color: #10b981;
  background-color: rgba(16, 185, 129, 0.03);
}

10. โฟกัสเฉพาะคีย์บอร์ด (:focus-visible)

จัดสไตล์ขอบเฉพาะเมื่อกดปุ่ม Tab บนคีย์บอร์ดมาโฟกัส (UX จะไม่แสดงสไตล์ขอบเวลากดคลิกเมาส์)

CSS:

/* แสดงขอบเรืองแสงเฉพาะตอนกด Tab มาโฟกัส */
.btn-focus-visible:focus-visible {
  outline: 3px solid #6366f1;
  outline-offset: 2px;
}

11. บังคับกรอก / ทางเลือก (:required & :optional)

จัดสไตล์ตัวแจ้งเตือนความสำคัญของช่องกรอกโดยอิงตาม Attribute จาก HTML อัตโนมัติ

CSS:

input:required + .field-badge::after { content: " *"; color: red; }
input:optional + .field-badge::after { content: " (ไม่บังคับ)"; }

12. ลูกโทนคนเดียว (:only-child)

สไตล์จะทำงานเมื่อ Element นั้นเป็นลูกคนเดียวในกล่อง (ไม่มีพี่น้องสิบลิ้งแท็กอื่นเลย)

ฉันคนเดียว 🙋‍♂️
พี่คนโต
น้องคนเล็ก

CSS:

.child-item:only-child {
  background-color: rgba(99, 102, 241, 0.2);
  border-color: #6366f1;
}

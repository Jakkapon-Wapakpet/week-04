# 🎨 CSS Cheat Sheet (คู่มือรวมคำสั่ง CSS สำคัญที่ใช้บ่อย)

คำสั่ง CSS (Properties) ในปัจจุบันมีอยู่หลายร้อยคำสั่งตามมาตรฐาน W3C แต่คำสั่งหลักๆ ที่นิยมและใช้งานบ่อยในการพัฒนาเว็บไซต์ สามารถแบ่งออกเป็นหมวดหมู่สำคัญๆ พร้อมคำอธิบายการใช้งานได้ดังนี้:

---

## 1. การจัดการข้อความและตัวอักษร (Typography & Text)

- `color`: กำหนดสีของข้อความ
- `font-family`: กำหนดรูปแบบหรือฟอนต์ของตัวอักษร
- `font-size`: กำหนดขนาดของตัวอักษร
- `font-weight`: กำหนดความหนา/บางของตัวอักษร (เช่น `bold`, `normal`, หรือตัวเลขเช่น `400`, `700`)
- `font-style`: กำหนดรูปแบบตัวเอียง (เช่น `italic`, `normal`)
- `text-align`: จัดตำแหน่งข้อความ (ชิดซ้าย `left`, ชิดขวา `right`, กึ่งกลาง `center`, หรือกระจายเต็มบรรทัด `justify`)
- `text-decoration`: ตกแต่งข้อความเพิ่มเติม เช่น ขีดเส้นใต้ (`underline`), ขีดฆ่า (`line-through`), หรือเอาเส้นออก (`none`)
- `text-transform`: แปลงตัวพิมพ์ใหญ่-เล็ก (เช่น ตัวพิมพ์ใหญ่ทั้งหมด `uppercase`, ตัวพิมพ์เล็ก `lowercase`)
- `line-height`: กำหนดระยะห่างระหว่างบรรทัด
- `letter-spacing`: กำหนดระยะห่างระหว่างตัวอักษร
- `word-spacing`: กำหนดระยะห่างระหว่างคำ

---

## 2. การจัดการกล่องและพื้นที่ (Box Model: Margin, Padding, Border)

- `width`: กำหนดความกว้างของ element
- `height`: กำหนดความสูงของ element
- `max-width` / `min-width`: กำหนดความกว้างสูงสุดและต่ำสุด
- `max-height` / `min-height`: กำหนดความสูงสูงสุดและต่ำสุด
- `padding`: กำหนดระยะห่างด้านใน ระหว่างเนื้อหากับขอบ (Border)
- `margin`: กำหนดระยะห่างด้านนอก ของ element เพื่อเว้นพื้นที่จาก element อื่น
- `border`: กำหนดเส้นขอบของ element (รวมความหนา รูปแบบ และสีไว้ในคำสั่งเดียว)
- `border-radius`: ทำมุมขอบให้โค้งมน
- `box-shadow`: ใส่เงาให้กับกล่องหรือ element
- `box-sizing`: กำหนดคำนวณขนาดกล่อง (เช่น `border-box` เพื่อรวม padding และ border ไว้ใน width/height)

---

## 3. การจัดวางเลย์เอาต์ (Layout & Display)

- `display`: กำหนดพฤติกรรมการแสดงผล เช่น บล็อก (`block`), บรรทัดเดียวกัน (`inline`), ซ่อน (`none`), หรือเปิดใช้งาน Flexbox (`flex`) และ Grid (`grid`)
- `position`: กำหนดรูปแบบการจัดวางตำแหน่ง (เช่น อยู่กับที่ `static`, สัมพัทธ์ `relative`, สัมบูรณ์ `absolute`, ยึดติดหน้าจอ `fixed`, หรือเกาะติดเมื่อเลื่อน `sticky`)
- `top` / `bottom` / `left` / `right`: กำหนดระยะห่างร่วมกับคำสั่ง `position`
- `z-index`: กำหนดลำดับชั้นทับซ้อนของ element (ยิ่งเลขมาก ยิ่งแสดงผลทับด้านบน)
- `overflow`: ควบคุมการแสดงผลเมื่อเนื้อหาล้นกล่อง (เช่น ซ่อน `hidden`, เลื่อนดูแถบ `scroll`, หรือแสดงอัตโนมัติ `auto`)
- `visibility`: ซ่อนหรือแสดง element แต่ยังคงพื้นที่เดิมไว้ (`visible` หรือ `hidden`)

---

## 4. เฟล็กซ์บ็อกซ์และกริด (Flexbox & Grid Layout)

### Flexbox Properties (สำหรับจัดการเลย์เอาต์ 1 มิติ)
- `flex-direction`: กำหนดทิศทางของแกนหลัก (เช่น แนวนอน `row` หรือแนวตั้ง `column`)
- `justify-content`: จัดเรียง items ตามแนวแกนหลัก (เช่น ชิดซ้าย/ขวา, กระจายตัว `space-between`, กึ่งกลาง `center`)
- `align-items`: จัดเรียง items ตามแนวแกนรอง (เช่น ชิดบน, ชิดล่าง, กึ่งกลาง `center`)
- `flex-wrap`: กำหนดให้ items ขึ้นบรรทัดใหม่เมื่อพื้นที่ไม่พอ (`wrap` หรือ `nowrap`)
- `gap`: กำหนดระยะห่างระหว่าง items ใน Flex หรือ Grid

### Grid Properties (สำหรับจัดการเลย์เอาต์ 2 มิติ ทั้งแถวและคอลัมน์)
- `grid-template-columns`: กำหนดจำนวนและความกว้างของคอลัมน์ใน Grid
- `grid-template-rows`: กำหนดจำนวนและความสูงของแถวใน Grid
- `grid-column` / `grid-row`: กำหนดพื้นที่การครอบครองพื้นที่ของ item ในตาราง Grid

---

## 5. สี พื้นหลัง และรูปภาพ (Colors & Backgrounds)

- `background-color`: กำหนดสีพื้นหลังของ element
- `background-image`: กำหนดรูปภาพพื้นหลัง
- `background-size`: กำหนดขนาดของรูปภาพพื้นหลัง (เช่น เต็มพื้นที่ `cover`, พอดีกรอบ `contain`)
- `background-position`: กำหนดตำแหน่งของรูปภาพพื้นหลัง
- `background-repeat`: กำหนดการทำซ้ำของรูปภาพพื้นหลัง (เช่น ไม่ให้ซ้ำ `no-repeat`)
- `opacity`: กำหนดความโปร่งใสของ element (ค่าตั้งแต่ 0 ถึง 1)

---

## 6. ลูกเล่นและการเคลื่อนไหว (Transitions & Animations)

- `transition`: กำหนดการเปลี่ยนแปลงแบบค่อยเป็นค่อยไป เมื่อค่า CSS เปลี่ยนแปลง (เช่น เมื่อเอาเมาส์ชี้ `:hover`)
- `animation`: กำหนดภาพเคลื่อนไหวแบบซับซ้อนโดยใช้ร่วมกับ `@keyframes`
- `transform`: แปลงรูปทรง ขยายหมุน หรือย้ายตำแหน่ง (เช่น หมุน `rotate()`, ขยาย `scale()`, เลื่อน `translate()`)
- `cursor`: เปลี่ยนรูปแบบของเคอร์เซอร์เมาส์เมื่อชี้โดน element (เช่น รูปมือ `pointer`, ตัวไอแบม `text`)

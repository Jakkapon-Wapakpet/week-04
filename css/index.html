📚 สรุปความรู้เรื่อง flex-flow

flex-flow คือคำสั่งลัด (Shorthand Property) ที่ช่วยลดรูปการเขียน CSS ของ Flex Container จาก 2 บรรทัดให้เหลือบรรทัดเดียว:

แบบเดิม (2 บรรทัด) 
.container {
  display: flex;
  flex-direction: row;
  flex-wrap: wrap;
}
แบบสั้นด้วย flex-flow (1 บรรทัด) 
.container {
  display: flex;
  flex-flow: row wrap;
}

ไวยากรณ์ (Syntax)

flex-flow: <flex-direction> <flex-wrap>;

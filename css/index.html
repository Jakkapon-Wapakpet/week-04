📚 สรุปความรู้เรื่อง flex-flow

flex-flow คือคำสั่งลัด (Shorthand Property) ที่ช่วยลดรูปการเขียน CSS ของ Flex Container จาก 2 บรรทัดให้เหลือบรรทัดเดียว:

แบบเดิม (2 บรรทัด) 
.container {
  display: flex;
  flex-direction: row;
  flex-wrap: wrap;
}
แบบสั้นด้วย flex-flow (1 บรรทัด) 
.container {
  display: flex;
  flex-flow: row wrap;
}

ไวยากรณ์ (Syntax)

flex-flow: <flex-direction> <flex-wrap>;

🔗 ผลกระทบของ flex-flow ต่อคำสั่ง Flexbox อื่น ๆ

การเปลี่ยนค่าใน flex-flow (โดยเฉพาะ flex-direction และ flex-wrap) ส่งผลโดยตรงต่อการทำงานของคำสั่ง Flexbox ตัวอื่น ๆ ดังนี้:

🔄

1. สลับแกนหลักของ justify-content และ align-items

  • ถ้า row (แนวนอน): justify-content จัดแนวนอน / align-items จัดแนวตั้ง
  • ถ้า column (แนวตั้ง): แกนจะสลับกัน! justify-content จะไปจัดแนวตั้งแทน ส่วน align-items จะไปจัดแนวนอน
📦

2. มีผลต่อคำสั่ง align-content

  • คำสั่ง align-content (ระยะห่างระหว่างบรรทัด) จะทำงาน เฉพาะเมื่อ flex-wrap เป็น wrap หรือ wrap-reverse เท่านั้น
  • ถ้าตั้งค่า flex-wrap: nowrap; คำสั่ง align-content จะไร้ผลทันที!
📏

3. ควบคุมมิติของ flex-basis ในตัวลูก

  • ถ้าทิศทางเป็น row: flex-basis จะส่งผลต่อความกว้าง (Width)
  • ถ้าทิศทางเป็น column: flex-basis จะส่งผลต่อความสูง (Height) แทน
